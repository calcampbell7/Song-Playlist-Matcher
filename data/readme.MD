# data dir structure

Ensure data dir looks like the following:
```
project-root/
├── data/
│   └── kagle/
│       ├── 1/
│       │   └── data/mpd.slice.*.json        # Playlist data (MPD slices)
│       └── 3/
│           └── SpotifyAudioFeaturesApril2019.csv  # Audio features CSV
│   └── processed_data/
```
